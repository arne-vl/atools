# `atools` - AVL CLI Tools

## Building manually
1. Make sure you have Go installed. You can download it from [golang.org](https://golang.org/dl/).
2. Clone the repository:
   ```bash
    git clone git@github.com:avl-systems/atools.git
    cd atools
   ```
3. Build the project:
    ```bash
    go mod tidy
    go build
    go install
    ```
